== Description ==

Geschaft Business is a multipurpose theme with modern layout with stylish, unique retina ready design. The moment you watch its demo, you’d absolutely love it. This theme is perfect for small to large businesses, it is translation ready making it perfect for you to expand your business. It has minimal and optimized code making this theme SEO friendly. This theme is suitable for small or large businesses, consultant, corporate agency, freelancer agency, business promotion, Industries, fashion, Online Shop, Magazine Publishers and many more. It has responsive design making it mobile friendly. The elegant Call to Action Button (CTA) is provided to grab attention of your customer. It has scores of customization option available to create layout as per your desire. It has testimonial section enabling you to add user experience over your website. You can set beautiful social media option for social sharing.

== Installation ==

Step 1: Add new page as "Home" select page attribute as "custom home page" publish it.

Step 2: Go to Customizer >> Homepage settings >> select static page >> and then select the home page >> publish it.

Step 2: GO to Dashboard >> Appearance >> Custiomize >> Theme Settings >> Here you wil get all the settings to setup home page.

Home page setup

Step 1. Add the new page >> Select page attribute "Custom Home Page" >> publish it.

Step 2. Go to Dashboard >> Customizer >> Homepage settings >> static page >> select home page >> publish it.

Slider setup

Step 1. Add new page >> Add title , content and featured images >> publish it.

Step 2. Go to Dashboard >> Customizer >> Top Banner >> select banner page >> publish it.

Services Setup

Step 1. Add new post >> Add title ,content, featured images, select same category >> publish it.

Step 2. Go to Dashboard >> Customizer >> Services >> Select category >> publish it.
